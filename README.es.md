## 🎹 Resumen
OveTyp_ es una aplicación de teclado sencilla diseñada para hacer que la práctica de mecanografía sea más entretenida. A medida que escribes correctamente y aciertas palabras, se añaden progresivamente instrumentos electrónicos a la pista de fondo, creando una interpretación musical sincronizada.

## ✨ Características Principales
- **Descubrimiento Instrumental**: Mantén tu ritmo para desbloquear nuevas capas de música electrónica en tiempo real.
- **Modo Práctica**: Niveles dedicados enfocados en zonas específicas del teclado (Índices, Medios, Anulares, Meñiques).
- **Visuales Audio-Reactivos**: Una MorphSphere 3D que reacciona a las bandas de frecuencia de la música y a tu velocidad de escritura.

## 🛠️ Stack Tecnológico
Construido para rendimiento y escalabilidad:
- **Core**: React 18 & TypeScript
- **Herramienta de Construcción**: Vite
- **Gráficos**: Three.js (WebGL) & Web Workers para física
- **Audio**: Web Audio API (Capas de pistas procedurales)
- **Arquitectura**: Lógica modular siguiendo principios de Arquitectura Hexagonal
